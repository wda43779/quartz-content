```bash
# The same command handles any compression format! Ex:

tar xf archive.tar.xz  # for .tar.xz files
tar xf archive.tar.gz  # for .tar.gz files
tar xf archive.tar     # for .tar files
```

`tar xf`解决大部分Linux平台见到的格式。
